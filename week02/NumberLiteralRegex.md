匹配Number Literal所有类型的正则

``` 
let reg = /^[\+\-]?(0\.?$|0[bB][01]+$|0[oO][0-7]+$|0[xX][0-9a-fA-F]+$|([1-9]\d*\.?\d*|0?\.\d+)([eE][\+\-]\d+)?$/;

```
解析：
```
^[\+\-]?  				  正负号开头或者无
0\.?$					      0 或者0.
0[bB][01]+          二进制
0[oO][0-7]+         八进制
0[xX][0-9a-fA-F]+		十六进制
[1-9]\d*\.?\d*			整数或带小数的浮点数
0?\.\d+					    小数点前为0或者无数字
([eE][\+\-]\d+)?		是否有科学计数法
```
